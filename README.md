> `astrbot_plugin_roleplay` v1.0（手动推进版）

随机生成完整场景卡——世界观、场景、身份、外貌、关系、开场白。把场景卡发给你的AI伴侣，ta来演。每次都不一样。

**核心设计：插件出数据，AI出灵魂。**

---

## 指令

| 命令 | 说明 |
|------|------|
| `角色设定 男/女` | 设定玩家性别（只需一次） |
| `角色扮演` | 完全随机生成场景卡 |
| `角色扮演 古风` | 限定风格 |
| `推进` | 时间流逝 + 场景变化旁白 |
| `加戏` | 随机突发事件 |
| `旁白` | 氛围描写 |
| `换场景` | 保留角色，换个地方 |
| `换剧情` | 同场景同角色，换关系和开场 |
| `存档` | 保存当前设定 |
| `读档` | 查看/读取存档 |

## 世界观

8个风格，每个都有独立的场景池、角色池：

古风 · 现代 · 校园 · 民国 · 科幻 · 奇幻 · 末日 · 江湖

## 数据规模

- 8 个世界观
- 78 个场景
- 108 个角色身份（男女各54）
- 12 种发型 × 10 种眼睛 × 10 种气质 × 12 种细节 = 14400 种外貌组合
- 15 种关系
- 10 种开场白
- 12 种推进事件
- 12 种突发事件
- 12 种旁白

理论组合数：接近无限。每次都是新故事。

## 安装

1. 将文件夹放入 `AstrBot/data/plugins/`
2. 重启或重载插件
3. 发 `角色设定 女` 然后 `角色扮演` 测试
